# vm22learnexpress

## install

`$ npm i`

# run

`$ node server.js`

## run with nodemon

`$ npx nodemon server.js`

või

`$ npm run start`

või (soovitatud)

`$ npm start`

Iga lehe alguses on meil vaja olnud terve html baas layout kopeerida.
Selleks, et suurte asjade koguaegsest kopeerimisest hoidud, kasutame nunjucks.
